# Adaptive Memory — Quick Start

1. **Add the hook** — Run `./install.sh` and add the printed JSON to `~/.openclaw/openclaw.json` (path = this folder’s `hook.js`).
2. **Restart** — `openclaw gateway restart`.
3. **Test** — New session, send a message; check `~/.openclaw/memory/YYYY-MM-DD.md` for "## Adaptive Memory Context (auto-injected)".

**Config:** `config.json` (see **SKILL.md** for options). **Install from ClawHub / publish:** **dist/adaptive-memory/INSTALL.md**. **Overview:** **README.md**.
